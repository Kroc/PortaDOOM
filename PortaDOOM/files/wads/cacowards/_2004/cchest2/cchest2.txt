Advanced engine needed  : Boom compatible

Title                   : Community Chest 2

Filename                : CChest2.wad

Release date            : December 2004

Author                  : Various Authors (See below)

Description             : Following the success of the original Community
			  Chest, Community Chest 2 is a 32 map megawad
			  including levels made by 26 different authors from
			  the Doom Community. Originally announced almost one
			  year ago in December 2003, it is now complete
			  and ready for your enjoyment.

===========================================================================
* What is included *

New levels              : 32
Sounds                  : No
Music                   : Yes
Graphics                : Yes
Dehacked/BEX Patch      : No
Demos                   : No
Other                   : No


* Play Information *

Game                    : DOOM2
Map #                   : MAP01-32
Single Player           : Designed for
Cooperative 2-4 Player  : Designed for
Deathmatch 2-4 Player   : Player starts only
Difficulty Settings     : Yes


* Construction *

Base                    : New from scratch
Build Time              : 1 Year
Editor(s) used          : Various
Known Bugs              : Map 32 has a segment violation in ports that haven't
			  increased the SEG limit
			  Map 23 has 2 mandatory HOMs in Eternity and some other
			  ports. These are mandatory because fixing them ruins
			  a couple of visual effects in other ports.
May Not Run With...     : Non-Boom compatible source ports and ports that do not
			  support Voodoo Dolls
