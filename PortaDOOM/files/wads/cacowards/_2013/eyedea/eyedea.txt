* Play Information *

Game                    : Doom 2
Map #                   : MAP01
Single Player           : Player starts only
Cooperative 2-4 Player  : No
Deathmatch 2 Player     : Designed for
Deathmatch 3+ Player    : Possible, but cramped
Other game styles       : None
Difficulty Settings     : ITYTD/HNTR - Newschool item layout.  Fewer
                                       health pickups, no stacked GA.
                          HMP - FFA layout, no stacked GA, BFG replaced
                                with Plasma Rifle.
                          UV/NM - Original Oldschool item layout.


* Construction *

Base                    : New from scratch
Build Time              : The layout was cranked out in an afternoon,
                          detailed in an evening, and refined over the
                          course of a week or so.
Editor(s) used          : Doom Builder 2, SLADE 3, GIMP 2.8
Known Bugs              : None known.  Tested extensively in Odamex and
                          Zandronum.  Seems to work in WinMBF as well.
May Not Run With...     : doom2.exe - The map makes use of several Boom
                          line types.
