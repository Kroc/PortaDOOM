[1] INTRODUCTION :

[1.1 Compatibility]

  3 Heures d'Agonie 3 is designed to be vanilla compatible so it should run
  virtually, if not, on every port. Doom2.wad is a definite prerequisite.

[1.2 Resources & Musics]

   (Check map information for more details)


[2] INSTALLATION :
------------------

  Simply either drag drop 3ha3.wad onto a source port executable (i.e. Zdoom or Prboom).
  Alternatively you can type into the command line:

  " 3ha3.exe -iwad doom2.wad -file 3ha3.wad " on DOS.
  or " chocolate-doom -iwad doom2.wad -file 3ha3.wad "
  
  If you aren't using chocolate-doom, replace chocolate-doom with the name of your source port of choice.

   Engines without internal DEHACKED support that DO support DeHackEd 
   (such as Chocolate Doom or the old PrBoom v2.02) need the -deh parameter 
   to load the DeHackEd patch.

  Engines that do not support DeHackEd (including Doom95, that is included
  in various releases of Final DOOM) can also run the add-on, but without any
  in-game story texts.
