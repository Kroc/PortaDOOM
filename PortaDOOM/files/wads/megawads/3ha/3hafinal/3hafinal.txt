Authors                 : JC

Email Address           : superjc@free.fr

General Description     : 3 Heures d'agonie is a collection of 32 vanilla /
			limit-removing speedmaps designed by members of the 
			French Doom Community. The general rule was that 
			each map was to be made within a time limit of 3 hours.

===========================================================================
* What is included *

New levels              : 32 maps
Sounds                  : no
Music                   : Yes
Graphics                : Yes
Dehacked/BEX Patch      : Yes
Demos                   : Yes (in order of appearance : Oxyde, JC, Oxyde & JC)
Other                   : no
Other files required    : Doom2.wad


* Play Information *

Game                    : Doom II Hell on Earth
Map #                   : Map01 ==> Map32
Single Player           : Designed for
Cooperative 2-4 Player  : Designed for
Deathmatch 2-4 Player   : Implemented but not optimal
Other game styles       : none
Difficulty Settings     : no


* Construction *

Base                    : New from scratch
Build Time              : 1 year and a half
Editor(s) used          : DooM Builder, eXtendable Wad Editor
Known Bugs              : see "Maps information"
May Not Run With        : heretic
Tested With             : PrBoom 2.4.7 / 2.5.0, Zdaemon 1.09, doom2.exe and doom2plus.exe
